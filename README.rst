txsyncml
========

Limited SyncML 1.1 server. Mostly suitable for doing one way backups
from a device to a server.

|travis|_ |coveralls|_

::

    $ virtualenv ve
    (ve)$ pip install -e .
    (ve)$ twistd txsyncml --help
